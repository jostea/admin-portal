Welcome!

You accessed Admin Panel, which allows to manage internship tests.

Here are some brief instructions.


Users

You can manage users to access the Admin Panel. Users with "SuperAdmin" role can create simple Admins and activate/deactivate them.

Disciplines

You can add, edit, view and delete disciplines.

Streams

You can add, edit, view and delete streams. Please, pay attentions that each stream schould be attached to an already existing discipline.

Tasks

You can create, view, modify and disable tasks. Tasks mean questions.

There are 3 types of questions:

  1. Standard Tasks
  2. Sql Tasks
  3. Code Tasks

Skills

You can manage different kinds of skills, which are required for a stream and discipline and should be evaluated.

Test Structure

Test Structure is a pattern (or template) of the test for a respective stream.

System randomly generates a set of questions from the tasks pool based on the test structure provided.

Report

In the Report you can review the answers provided by a candidate and change its status (correct/incorrect) if necessary.
