Welcome!

You accessed Admin Panel to manage interview tests and questions.

Here are some brief instructions.


Users

You can manage users to access the Admin Panel. Users with "SuperAdmin" role can create simple Admins and activate/deactivate them.

Disciplines

You can add, edit, view and delete disciplines.

Streams

You can add, edit, view and delete streams. Please, pay attentions that each stream schould be attached to an already existing discipline.

Tasks

You can create, view, modify and disable tasks. Tasks mean questions.

There are 3 types of questions:

  1. Standard Tasks
  2. Sql Tasks
  3. Code Tasks
